Emote - The Text Analyzer

A sleek and expressive Android app with dynamic theming, intuitive navigation, and Firebase-powered user experience.

View Code

Project Overview

Emote is a Kotlin-based Android application built using Jetpack Compose. It features a modern, responsive UI with light and dark theme support, bottom navigation for seamless screen transitions, and Firebase Authentication to manage user login and profile data. The app leverages Compose's declarative approach to deliver an elegant and customizable user interface with Google Fonts integration and animated components. The app does the following:

  • Analyzes text input to detect emotions (joy, anger, sadness, etc.) using an AI API
  • Tracks emotional history with a local database (Room)
  • Supports authentication (Firebase) and theme customization (light/dark mode)
  • Features a clean, Material 3-compliant UI with custom typography and colors

Design and Development Process

Research and Planning

  • Selected Firebase Auth for simplicity and scalability
  • Used Material Theme Builder for color schemes supporting light/dark modes
  • MVVM pattern with ViewModels managing UI state
  • Repository pattern to abstract data sources (Firebase, Room, API)
  • Chose the Emotion Analysis API to build the project with

Challenges and Solutions

  • Bottom navigation failed to track current route: Used currentBackStackEntryAsState() and rememberNavController() for accurate route tracking.
  • Theme toggle did not reflect in real-time: Refactored using StateFlow and collectAsStateWithLifecycle() for lifecycle-aware state updates.
  • Firebase user data was inaccessible inside Composables: Exposed user data via LoginViewModel and collected it using Compose state.
  • Custom fonts were not applying consistently: Integrated Google Fonts using Compose Typography and FontFamily.
  • Profile picture loading caused crashes: Switched to Coil's rememberAsyncImagePainter to load and fallback safely.

Final Outcome

The current version of Emote successfully delivers a polished mobile experience with a fully Compose-based UI, Firebase-backed authentication, and seamless theme switching. It sets the foundation for future features like profile editing, onboarding, DataStore-based theme persistence, and real-time data syncing.
