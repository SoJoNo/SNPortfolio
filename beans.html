Beans - A Coffee Site

Beans is a Ruby on Rails application that helps coffee enthusiasts discover Starbucks stores and premium coffee products worldwide.

Project Overview

Beans was made for one of my school projects. It was the Introduction to Rails project and it mainly asked for working with tables and seeding databases. This was my first big project with Rails and because of that I faced a couple challenges like where to even start with the code.

Design and Development Process

Research and Planning

Entity Relationship Diagram

When starting my research, I had a couple issues finding an API I could actually work with. I wanted to do something coffee related for sure but there weren't any APIs that matched what I wanted. I wanted something with just generic coffee stores not Starbucks related but there were none and I was crunched for time so I couldn't make mine so I settled for the Starbucks Location Worldwide API on Kaggle. Then, I also wanted something for specific coffee types and brands. Thankfully, this was a lot easier to achieve as I found the Coffee Reviews Dataset also on Kaggle

Challenges and Solutions

The real challenge came when implementing the review system:
Getting ratings to properly update across different models
Making sure all the filters worked smoothly
Debugging those frustrating nil errors when users weren't properly associated


Through trial and error (and some serious StackOverflow digging), I implemented solutions like:
Smart rating averages that update in real-time
A flexible filtering system that works across both shops and brands
Proper error handling for anonymous reviews

Final Outcome

I built a fully-functional coffee discovery app that:
Displays coffee shops worldwide with interactive maps
Shows detailed coffee brand information with mock user reviews
Features smart filtering for both shops and coffee products


The project taught me valuable lessons about:
Working with imperfect datasets
Building intuitive forms
Managing database relationships
Problem-solving through debugging
